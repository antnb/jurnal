<p><span class="font1" style="font-weight:bold;font-style:italic;">Abstract</span></p>
<p><span class="font1" style="font-style:italic;">Almost everyone looks at reviews before deciding to buy an item in e-commerce. Consumers say that online reviews influence their purchasing decisions. Based on these data, consumers need sentiment reviews to make a decision to choose a product/service. However, the results of the sentiment analysis are still less specific, so the review classification process is carried out based on the review category. Sentiment classification process based on clothing category is carried out using the Convolutional neural network method. The amount of data used is 3384 data with 3 categories. The category classification model shows good performance. When evaluated with testing data (unseen data), the accuracy value is 88%, the precision value is 88%, recall is 88% and the f1-score is 88%. For the sentiment classification model with the bottoms category, the resulting accuracy value is 80%, precision is 81%, recall is 80%, and f1-score is 79%. For the sentiment classification model with the dresses category, the accuracy value is 81%, precision is 81%, recall is 81%, and f1-score is 81%. For sentiment classification with the tops category the resulting accuracy value is 77%, precision is 77%, recall is 77%, and f1-score is 77%.</span></p>
<p><span class="font1" style="font-weight:bold;font-style:italic;">Keywords: e-commerce, review, sentiment, NLP, CNN</span></p>
<ul style="list-style:none;"><li>
<h2><a name="bookmark2"></a><span class="font1" style="font-weight:bold;"><a name="bookmark3"></a>1. &nbsp;&nbsp;&nbsp;Pendahuluan</span></h2></li></ul>
<p><span class="font1">Beberapa pengguna e-commerce dalam menentukan membeli suatu produk umumnya melihat ulasan sebelum memutuskan untuk membeli barang, tepatnya 91% seseorang yang berusia 18-34 tahun memercayai ulasan online seperti halnya rekomendasi pribadi, dan 93% konsumen mengatakan bahwa ulasan online mempengaruhi keputusan pembelian mereka [1]. Berdasarkan data tersebut, konsumen membutuhkan ulasan sentimen guna membuat keputusan untuk memilih suatu produk/jasa. Namun hasil analisis sentiment yang dihasilkan masih kurang spesifik, maka dilakukan proses klasifikasi ulasan tersebut berdasarkan kategori ulasan. Sehingga analisis sentimen yang dihasilkan lebih spesifik. CNN biasa digunakan untuk mendeteksi dan mengenali objek pada sebuah gambar. CNN juga dapat diaplikasikan sebagai Natural Language Processing dengan mengubah representasi dari sebuah kalimat menjadi vektor. Implementasi CNN dalam NLP dapat menghasilkan performa yang baik. Seperti pada penelitian Rahman [2] hasil evaluasi untuk data opini publik pada Instagram mendapatkan hasil rata-rata precision 95.80%, recall 88.12%, f1-score 91.62%.</span></p>
<p><span class="font1">Dataset E-Commerce pakaian yang digunakan terbagi menjadi dua sentimen yaitu positif dan negatif. Pada penelitian ini melakukan analisis sentimen berdasarkan kategori yang terdapat di dalam dataset seperti tops, bottoms, jackets, dresses, dan intimates. Penelitian ini dilakukan dengan menggunakan dataset berbahasa Inggris yang diperoleh dari Kaggle dengan total keseluruhan data sebanyak 23.485 data dengan perbandingan data positif dan negatif 19314 data dengan label positif dan 4172 data dengan label negatif.</span></p>
<ul style="list-style:none;"><li>

Data tersebut kemudian melalui </span><span class="font1" style="font-style:italic;">preprocessing</span><span class="font1">. <p><span class="font1" style="font-weight:bold;">2.2. &nbsp;</span><span class="font1" style="font-weight:bold;font-style:italic;">Preprocessing</span></p></li></ul>
<p><span class="font1">Sebelum </span><span class="font1" style="font-style:italic;">dataset</span><span class="font1"> diolah menggunakan metode </span><span class="font1" style="font-style:italic;">Convolutional Neural Network, dataset</span><span class="font1"> melalui tahapan </span><span class="font1" style="font-style:italic;">pre-processing</span><span class="font1"> yang mana pada tahapan </span><span class="font1" style="font-style:italic;">pre-processing</span><span class="font1"> terdapat beberapa proses didalamnya seperti </span><span class="font1" style="font-style:italic;">tokenization, case folding, text to sequences,</span><span class="font1"> dan </span><span class="font1" style="font-style:italic;">filtering.</span><span class="font1"> Tahap ini dilakukan untuk memastikan data yang akan digunakan telah bersih, sehingga dapat mencegah </span><span class="font1" style="font-style:italic;">error</span><span class="font1"> atau turunnya performa mesin [3]. Setelah melewati tahapan </span><span class="font1" style="font-style:italic;">pre-processing</span><span class="font1"> data tersebut dikelompokkan berdasarkan pada kategori pada masing-masing data. Kemudian representasi data tersebut diubah menjadi vektor dengan menggunakan algoritma </span><span class="font1" style="font-style:italic;">word embedding</span><span class="font1">.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font1">a.</span><span class="font1" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Under sampling</span></p></li></ul>
<p><span class="font1">Proses ini merupakan proses untuk normalisasi jumlah data menjadi seimbang. Metode ini bekerja dengan mengurangi data mayoritas secara acak, sehingga jumlah data sama dengan data minoritas. Setelah melalui proses ini data yang digunakan sebanyak 3384 data dengan 3 kelas (</span><span class="font1" style="font-style:italic;">tops, bottoms, dresses)</span></p>
<ul style="list-style:none;"><li>
<p><span class="font1">b.</span><span class="font1" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Case folding</span></p></li></ul>
<p><span class="font1">Pada tahapan </span><span class="font1" style="font-style:italic;">case folding</span><span class="font1"> berfungsi untuk mengubah keseluruhan data teks yang ada dalam </span><span class="font1" style="font-style:italic;">dataset</span><span class="font1"> menjadi huruf kecil sehingga keseluruhan teks memiliki bentuk standar[3]</span></p>
<ul style="list-style:none;"><li>
<p><span class="font1">c.</span><span class="font1" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Filtering</span></p></li></ul>
<p><span class="font1">Pada tahap ini dilakukan proses menghilangkan simbol-simbol, </span><span class="font1" style="font-style:italic;">stopwords,</span><span class="font1"> dan angka pada kalimat. Sehingga tidak terdapat simbol, angka, dan stopwords dalam data yang digunakan.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font1">d.</span><span class="font1" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Tokenization</span></p></li></ul>

<ul style="list-style:none;"><li>
<p><span class="font1">e.</span><span class="font1" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Text to sequences</span></p></li></ul>
<p><span class="font1">Tahap ini dilakukan untuk merubah representasi kata dalam kalimat menjadi angka. Kata yang dirubah digantikan dengan index kata dalam korpus kata[4]</span></p>
<ul style="list-style:none;"><li>
<p><span class="font1">f.</span><span class="font1" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Splitting dataset</span></p></li></ul>
<p><span class="font1">Proses ini dilakukan dengan menggunakan metode holdout. Untuk pembagian dataset yaitu 90% data digunakan pada proses training dan 10% data digunakan untuk </span><span class="font1" style="font-style:italic;">testing</span><span class="font1">.</span></p>

<p><span class="font1">Setiap kata diubah menjadi vektor yang merepresentasikan sebuah titik pada space dengan dimensi tertentu. Kata-kata yang memiliki arti serupa memiliki representasi vektor dengan posisi yang tidak berjauhan antara satu dengan lainnya pada vektor space tersebut [5]</span></p>

<h2><a name="bookmark10"></a><span class="font1" style="font-weight:bold;"><a name="bookmark11"></a>2.4 &nbsp;&nbsp;&nbsp;Convolutional Neural Network (CNN)</span></h2></li></ul>
<p><span class="font1" style="font-style:italic;">Convolutional Neural Network</span><span class="font1"> merupakan perkembangan dari </span><span class="font1" style="font-style:italic;">Artificial Neural Network</span><span class="font1"> dengan penambahan </span><span class="font1" style="font-style:italic;">layer</span><span class="font1"> pada </span><span class="font1" style="font-style:italic;">hidden layer</span><span class="font1">. CNN umumnya yang digunakan untuk proses klasifikasi gambar. CNN dapat diaplikasikan dalam </span><span class="font1" style="font-style:italic;">natural language processing</span><span class="font1"> dengan mengubah representasi dari sebuah kalimat menjadi </span><span class="font1" style="font-style:italic;">matrix</span><span class="font1">. Dengan konfigurasi </span><span class="font1" style="font-style:italic;">Hyperparameter</span><span class="font1">dan vektor statis CNN dengan konfigurasi sederhana dapat menghasilkan hasil yang baik pada beberapa percobaan yang berbeda [6]. Arsitektur dari CNN yang akan diimplementasikan oleh penulis diambil berdasarkan referensi pada penelitian [2] Namun dengan sedikit perubahan pada jumlah units dan penambahan layer dropout setelah layer konvolusi dan layer max pooling. Setelah melalui proses word embedding data dimasukkan ke dalam lapisan konvolusi. Pada penelitian ini penulis akan menggunakan konvolusi 1D. Pemilihan konvolusi 1D dipilih dikarenakan ketika diaplikasikan dengan menggunakan teks sebagai data input, konvolusi 1D merupakan pilihan yang biasanya digunakan [7]</span></p>
<p><span class="font1">Sebuah kernel konvolusi (filter) berukuran tertentu digeser ke seluruh vektor tersebut. Pada setiap pergeseran menghasilkan </span><span class="font1" style="font-style:italic;">dot product</span><span class="font1">. Dengan menggeserkan filter tersebut ke seluruh vektor maka dihasilkan sebuah feature map. Setelah melalui lapisan konvolusi fitur aktivasi yang digunakan adalah ReLu. Output dari fungsi ReLu kemudian di masukkan ke dalam pooling layer. Untuk arsitektur yang digunakan pada penelitian ini dapat dilihat pada Gambar 2 dan Gambar 3. Arsitektur dari CNN yang akan diimplementasikan oleh penulis diambil berdasarkan referensi pada penelitian [2] dengan sedikit perubahan pada jumlah units dan penambahan layer dropout setelah layer konvolusi dan layer max pooling.</span></p>
<p><span class="font1">Setelah melalui proses word embedding data dimasukkan ke dalam lapisan konvolusi. Pada lapisan konvolusi terdapat proses perkalian </span><span class="font1" style="font-style:italic;">matrix</span><span class="font1"> yang menghasilkan </span><span class="font1" style="font-style:italic;">feature map</span><span class="font1">. Setelah melalui lapisan konvolusi fitur aktivasi yang digunakan adalah ReLu. Output dari fungsi ReLu kemudian di masukkan ke dalam pooling layer. Bagian berikutnya merupakan lapisan max pooling layer, lapisan ini digunakan untuk mereduksi input secara spasial dengan menggunakan operasi down-sampling. Penulis menggunakan metode max pooling layer. </span><span class="font1" style="font-style:italic;">Dropout</span><span class="font1"> ditambahkan setelah lapisan konvolusi dan lapisan </span><span class="font1" style="font-style:italic;">max pooling</span><span class="font1">. Penambahan parameter </span><span class="font1" style="font-style:italic;">dropout</span><span class="font1"> bertujuan untuk mengurangi persentase terjadinya </span><span class="font1" style="font-style:italic;">overfitting</span><span class="font1"> dan </span><span class="font1" style="font-style:italic;">underfitting</span><span class="font1">, selain itu juga mempercepat proses </span><span class="font1" style="font-style:italic;">learning</span><span class="font1">. Rentang nilai yang digunakan pada parameter </span><span class="font1" style="font-style:italic;">dropout</span><span class="font1"> berkisar dari 0 sampai 1. Pada penelitian ini parameter </span><span class="font1" style="font-style:italic;">dropout</span><span class="font1"> ditentukan secara eksplisit dengan rentang angka yang sesuai dengan data yang penulis miliki, untuk memberikan akurasi yang maksimal dari model tersebut dan dilakukan secara repetitif hingga ditemukan </span><span class="font1" style="font-style:italic;">Hyperparameter</span><span class="font1">terbaik. Lapisan </span><span class="font1" style="font-style:italic;">flattening</span><span class="font1"> ditambahkan untuk memastikan dimensi dari vektor yang dihasilkan sebelumnya sesuai dengan dimensi vektor yang dibutuhkan pada layer selanjutnya. Pada lapisan selanjutnya terdapat lapisan </span><span class="font1" style="font-style:italic;">fully-connected</span><span class="font1"> dengan jumlah units sebanyak 500 dan berdasarkan referensi yang diambil pada penelitian [2]. Pada output layer terdapat lapisan </span><span class="font1" style="font-style:italic;">fully-connected</span><span class="font1"> dengan jumlah </span><span class="font1" style="font-style:italic;">neuron</span><span class="font1"> tiga untuk mesin klasifikasi kategori dan fungsi aktivasi </span><span class="font1" style="font-style:italic;">softmax</span><span class="font1">. Sedangkan untuk mesin klasifikasi sentimen, pada output layerjumlah </span><span class="font1" style="font-style:italic;">neuron</span><span class="font1"> sebanyak dua dan fungsi aktivasi </span><span class="font1" style="font-style:italic;">sigmoid</span><span class="font1">.</span></p><img src="https://jurnal.harianregional.com/media/86503-2.jpg" alt="" style="width:310pt;height:95pt;">
