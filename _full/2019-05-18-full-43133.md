<h1><a name="bookmark0"></a><span class="font3" style="font-weight:bold;"><a name="bookmark1"></a>Implementation of Zoning and K-Nearest Neighbors in Character Recognition of Wrésastra Script</span></h1>
<p><span class="font2">I Wayan Agus Surya Darma</span></p>
<p><span class="font2">Informatics Engineering, STMIK STIKOM Indonesia Tukad Pakerisan 97<sup>th</sup> Denpasar, Bali, Indonesia </span><a href="mailto:surya@stiki-indonesia.ac.id"><span class="font2">surya@stiki-indonesia.ac.id</span></a></p>
<p><span class="font2" style="font-weight:bold;font-style:italic;">Abstract</span></p>
<p><span class="font2" style="font-style:italic;">Balinese script is an important aspect that packs the Balinese culture from time to time which continues to experience development along with technological advances. Balinese script consists of three types (1) Wrésastra, (2) Swalalita and (3) Modre which have different types of characters. The Wrésastra and Swalalita script are Balinese scripts which grouped into the script criteria that are used to write in the field of everyday life. In this research, the zoning method will be implemented in the feature extraction process to produce special features owned by Balinese script. The results of the feature extraction process will produce special features owned by Balinese script which will be used in the classification process to recognize the character of Balinese script. Special features are produced using the zoning method, it will divide the image characters area of Balinese scripts into several regions, to enrich the features of each Balinese script. The result of feature extractions is stored as training data that will be used in the classification process. K-Nearest Neighbors is implemented in the special feature classification process that is owned by the character of Balinese script. Based on the results of the test, the highest level of accuracy was obtained using the value K=3 and reference=10 with the accuracy of Balinese script recognition 97.5%.</span></p>
<p><span class="font2" style="font-weight:bold;font-style:italic;">Keywords</span><span class="font2" style="font-style:italic;">: Balinese Script, Feature Extraction, Classification, Zoning, KNN</span></p>

A study that provides an overview of the special feature extraction of Balinese script using the </span><span class="font2" style="font-style:italic;">zoning </span><span class="font2">method also has been carried out [3]. The results of this study can be used as a basis for the development of a Balinese script characters recognition system. Other researches on the implementation of KNN have also been carried out [4] for the classification of Batik Parang motifs so that batik types can be identified based on the motif features produced. Other research on KNN for the introduction of Arabic characters has also been done [5]. This study implements KNN in the classification of Arabic characters.</span></p>
<p><span class="font2">The research that related to the introduction system has been carried out in the introduction of the palm entitled &quot;</span><span class="font2" style="font-style:italic;">High Performance Palm print Identification System Based on Two Dimensional Gabor</span><span class="font2">&quot; [6] resulting in a success rate of 98.7% aims to introduce the palm ROI segmentation method at the center of the two-stage moment and apply the two-dimensional Gabor method to produce the palm code as a palm feature and use the </span><span class="font2" style="font-style:italic;">hamming</span><span class="font2"> distance method to measure the similarity level of two palm vectors. Previous research related to KNN which uses a custom image to train the classifier [8] to recognize handwritten or printed text. Other research about feature extraction to extract color image using texture methods [11] which these features can be used as a primary key to identify and recognize the image. Other research related to character recognition on Hindi Optical Character Recognition for Printed Documents using KNN [14] and a research about Javanese handwritten character classification [15].</span></p>
<p><span class="font2">The character of Balinese script that is used as primary data is the image characters of Balinese script from handwriting. In this research, the zoning method will be implemented in the feature extraction process to produce the special feature of Balinese script. The result of feature</span></p>
<p><span class="font2">extraction process will produce special features of Balinese script that will be used in the classification process to recognize the characters of Balinese script.</span></p>
<p><span class="font2">Special features is produced using the zoning method, it will divide the image characters area of Balinese scripts into several regions, to enrich the features of each Balinese script. The result of feature extractions are stored as training data that will be used in the classification process. KNN is implemented in the special feature classification process that is owned by the character of Balinese script. In the classification process, the character feature of Balinese script will be classified and matched with the Balinese script features stored in the training data. The results of the introduction will be presented on the percentage of the nearest neighbors, in accordance with the similarity of the characteristics of the Balinese script with training data that has been through the classification process using KNN.</span></p>

<h2><a name="bookmark6"></a><span class="font2" style="font-weight:bold;"><a name="bookmark7"></a>2.1 &nbsp;&nbsp;&nbsp;Data Training Process</span></h2></li></ul>
<p><span class="font2">The implementation of the zoning and KNN methods is carried out at each of the character recognition stages in </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script. In the first stage, there is a process of forming training data that is used as a reference in the introduction stage. During the training data formation phase, the character image of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script through the feature extraction process. At this stage, the zoning method is implemented to divide the character image of the Balinese script. The division of regions in the image of Balinese script aims to provide special features variations on each character image of Balinese script. The results of this process will go through the classification stage and stored into the training data model. In this research, the formation of</span></p>
<p><span class="font2">training data models can be determined based on the number of training data for each character image of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script</span></p>
<ul style="list-style:none;"><li>
<p><span class="font2">a. &nbsp;&nbsp;&nbsp;Training Image</span></p></li></ul>
<p><span class="font2">The first process is to register the training image to be formed into a training model. This process aims to determine the character image of Balinese script that will be used as training data. In this study, the variation of training image used for each Balinese script are 5, 7 and 10 images, which means the total number of images used as training data for 27 </span><span class="font2" style="font-style:italic;">Wrésastra </span><span class="font2">Scripts are variation of 135, 189 and 270 images. Variations in the amount of training data for each </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script will be used as a training model for the testing phase for the recognition of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font2">b. &nbsp;&nbsp;&nbsp;Determine the Training Model</span></p></li></ul>
<p><span class="font2">The second process at this stage is the feature extraction process to acquire the special features of each Balinese script characters that are used as training data model. In this process, it can be created training data model based on the number of Balinese scripts that is used.</span></p>
<ul style="list-style:none;"><li>

The implementation of the zoning method in the feature extraction process is done in order to obtain unique features for each character. The zoning method is implemented by zoning on each character during the feature extraction process. The value of the feature extraction results through the KNN classification process, therefore the </span><span class="font2" style="font-style:italic;">Wrésastra </span><span class="font2">script feature data will be grouped. The classification results are stored as a training data model.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font2">2.2 &nbsp;&nbsp;&nbsp;Recognition Stage</span></p></li></ul>
<p><span class="font2">The second stage is the introduction stage of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script by implementing KNN. At this stage, the character image of Wrésastra script is tested, through the feature extraction process first to get the special feature values of the </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script character image tested. After the special features of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script are obtained, this feature will be classified with the training data model that has been formed using KNN.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font2">a. &nbsp;&nbsp;&nbsp;Image Acquisition</span></p></li></ul>
<p><span class="font2">The image that is used in the introduction process is the handwritten of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script characters. <ul style="list-style:none;"><li>
<p><span class="font2">b. &nbsp;&nbsp;&nbsp;Zoning Feature Extraction</span></p></li></ul>
<p><span class="font2">Feature extraction process to acquire specific features of each character of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script that is used as a training data model. The zoning method is implemented to divide the image area of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script, therefore the resulting features will be more varied. The implementation of zoning method in the feature extraction process is done in order to obtain unique features for each character. The stages in the feature extraction process at the introduction stage are the same as the Data Training Process stage.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font2">c. &nbsp;&nbsp;&nbsp;KNN Classification</span></p></li></ul>
<p><span class="font2">The classification process aims to classify features of </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script. The KNN method classifies </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script feature data based on the closest neighbors. The feature value of the </span><span class="font2" style="font-style:italic;">Wrésastra</span><span class="font2"> script test will be classified with feature values in the training data model that has been formed previously at the training process data stage.</span></p>
