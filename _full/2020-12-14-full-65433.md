<h1><a name="bookmark0"></a><span class="font4" style="font-weight:bold;"><a name="bookmark1"></a>Implementasi Algoritma PRNG pada Aplikasi Port Knocking Sebagai Perlindungan Server</span></h1>

<p><span class="font3" style="font-weight:bold;font-style:italic;">Abstrak</span></p>
<p><span class="font3" style="font-style:italic;">Keamanan jaringan harus dipertimbangkan dan dilindungi dengan baik. Masalah yang terjadi jika keamanan jaringan tidak terlindungi akan menyebabkan kerusakan sistem server dan dapat terjadinya akses tidak sah pada sistem server. Penelitian ini bertujuan untuk melakukan proses autentikasi jaringan antara komputer klien dengan komputer server dalam memvalidasi klien yang akan terhubung dengan sistem server. Penelitian dilakukan dengan metode autentikasi port knocking yang telah diimplementasikan algoritma PRNG (pseudo random number generator) dengan acuan seed waktu sistem. Hasil dari penelitian ini yaitu autentikasi port knocking dilakukan menggunakan hasil random sequence port yang berbeda sesuai dengan waktu sistem saat random number dibangkitkan guna untuk memperkuat autentikasi dalam memvalidasi klien yang akan terhubung dengan sistem server.</span></p>
<p><span class="font3" style="font-weight:bold;font-style:italic;">Kata kunci: </span><span class="font3" style="font-style:italic;">Keamanan jaringan, Autentikasi, Port Knocking, PRNG.</span></p>
<p><span class="font3" style="font-weight:bold;font-style:italic;">Abstract</span></p>
<p><span class="font3" style="font-style:italic;">Network security must be considered and protected properly. Problems occur if unprotected network security causes damage to the server system and unauthorized access to the server system can occur. This study aims to carry out a network authentication process between client computers and server computers in validating clients that will be connected to the server system. The study was conducted with the port knocking authentication method which has been implemented by the PRNG algorithm (pseudo-random number generator) with a system time seed reference. The results of this study are that port knocking authentication will be performed using different random port sequence results according to the system time when the random number is generated, to strengthen authentication in validating clients who will be connected to the server system.</span></p>
<p><span class="font3" style="font-weight:bold;font-style:italic;">Keywords : </span><span class="font3" style="font-style:italic;">Network Security, Authentication, Port Knocking, PRNG.</span></p>

<h2><a name="bookmark4"></a><span class="font3" style="font-weight:bold;"><a name="bookmark5"></a>1. &nbsp;&nbsp;&nbsp;&nbsp;Pendahuluan</span></h2></li></ul>
<p><span class="font3">Keamanan jaringan harus dipertimbangkan dan dilindungi dengan baik. Keamanan jaringan merupakan aktivitas dalam pengamanan perangkat </span><span class="font3" style="font-style:italic;">server</span><span class="font3">, klien dan media transmisi data. Jika keamanan jaringan tidak dipertimbangkan dan dilindungi dengan baik, maka peretas mungkin dapat menargetkan saluran komunikasi untuk mendapatkan data yang di-enkripsi, kemudian men-dekripsi dan memasukkan kembali pesan palsu [1],[2]. </span><span class="font3" style="font-style:italic;">Man-in-the-middle-attack </span><span class="font3">merupakan serangan jaringan dimana penyerang menyampaikan dan mengubah komunikasi antara dua pihak yang terpercaya bahwa mereka sedang berkomunikasi secara langsung antara satu sama lain [3]. Autentikasi diperlukan sebagai pembentukan identitas antara dua belah pihak yang berkomunikasi untuk membangun layanan komunikasi yang aman saat dilakukan pertukaran data [4]. </span><span class="font3" style="font-style:italic;">Port knocking</span><span class="font3"> merupakan sebuah metode autentikasi yang digunakan untuk menyembunyikan </span><span class="font3" style="font-style:italic;">service port</span><span class="font3">, beserta untuk membuka akses pada </span><span class="font3" style="font-style:italic;">service port</span><span class="font3"> tertutup harus menggunakan ketukan </span><span class="font3" style="font-style:italic;">port sequence</span><span class="font3"> [5]. </span><span class="font3" style="font-style:italic;">Pseudo random number generator </span><span class="font3">(PRNG) merupakan algoritma yang dapat menyiratkan rumus matematika, tabel perhitungan sederhana atau bahkan keduanya pada waktu tertentu dan menghasilkan angka acak berurutan [6],[7]. Melalui autentikasi </span><span class="font3" style="font-style:italic;">port knocking</span><span class="font3">, perangkat klien dapat mengakses </span><span class="font3" style="font-style:italic;">service</span></p>
<p><span class="font3" style="font-style:italic;">port</span><span class="font3"> tertutup seperti SSH, FTP dan MySQL pada perangkat </span><span class="font3" style="font-style:italic;">server</span><span class="font3">. Autentikasi yang digunakan pada penelitian ini yaitu menggunakan aplikasi Knockd dan Python [8].</span></p>
<p><span class="font3">Gagasan dalam implementasi algoritma PRNG pada aplikasi </span><span class="font3" style="font-style:italic;">port knocking</span><span class="font3"> dilakukan dengan tujuan untuk memperkuat proses autentikasi dalam memvalidasi komputer klien yang akan terhubung dengan sistem </span><span class="font3" style="font-style:italic;">server</span><span class="font3">. Dari beberapa penelitian yang telah dilakukan proses autentikasi masih menggunakan </span><span class="font3" style="font-style:italic;">static sequence port</span><span class="font3"> [9],[10] yang menyebabkan proses autentikasi masih cukup rentan dari serangan jaringan. Maka dari itu dilakukan pengembangan sebuah aplikasi yang dapat melakukan proses autentikasi menggunakan </span><span class="font3" style="font-style:italic;">random sequence port </span><span class="font3">untuk memperkuat proses autentikasi. </span><span class="font3" style="font-style:italic;">Random sequence port</span><span class="font3"> diperoleh dari pembangkitan nilai </span><span class="font3" style="font-style:italic;">random</span><span class="font3"> menggunakan acuan </span><span class="font3" style="font-style:italic;">seed</span><span class="font3"> waktu sistem. Selain itu, aplikasi dibangun menggunakan </span><span class="font3" style="font-style:italic;">framework</span><span class="font3"> PyQt4 (GUI) dengan tujuan untuk mempermudah pengguna dalam menggunakan aplikasi yang dirancang pada penelitian.</span></p>

<h2><a name="bookmark8"></a><span class="font3" style="font-weight:bold;"><a name="bookmark9"></a>3. &nbsp;&nbsp;&nbsp;Kajian Pustaka</span></h2></li></ul>

<h2><a name="bookmark10"></a><span class="font3" style="font-weight:bold;"><a name="bookmark11"></a>3.1. &nbsp;&nbsp;&nbsp;Python</span></h2></li></ul>
<p><span class="font3">Python merupakan bahasa pemrograman yang memungkinkan pengguna bekerja dengan cepat dan mengintegrasikan sistem dengan lebih efektif. <h2><a name="bookmark14"></a><span class="font3" style="font-weight:bold;"><a name="bookmark15"></a>3.3. &nbsp;&nbsp;&nbsp;Knockd</span></h2></li></ul>
<p><span class="font3">Knockd merupakan aplikasi yang digunakan sebagai </span><span class="font3" style="font-style:italic;">server port-knock</span><span class="font3">. Knockd mendengarkan semua lalu lintas pada antarmuka ethernet untuk mencari urutan &quot;ketukan&quot; khusus dari </span><span class="font3" style="font-style:italic;">port-hit</span><span class="font3">. Klien membuat </span><span class="font3" style="font-style:italic;">port-hit</span><span class="font3"> dengan mengirimkan paket TCP (atau UDP) menuju layanan </span><span class="font3" style="font-style:italic;">port</span><span class="font3"> pada </span><span class="font3" style="font-style:italic;">server</span><span class="font3">. </span><span class="font3" style="font-style:italic;">Port</span><span class="font3"> pada </span><span class="font3" style="font-style:italic;">server</span><span class="font3"> tidak perlu terbuka dikarenakan Knockd mendengarkan di tingkat lapisan </span><span class="font3" style="font-style:italic;">link-layer</span><span class="font3">. Knockd melihat semua lalu lintas meskipun ditujukan untuk </span><span class="font3" style="font-style:italic;">port</span><span class="font3"> tertutup. Ketika </span><span class="font3" style="font-style:italic;">server</span><span class="font3"> mendeteksi urutan tertentu dari </span><span class="font3" style="font-style:italic;">port-hit</span><span class="font3">, </span><span class="font3" style="font-style:italic;">server</span><span class="font3"> akan menjalankan perintah yang ditentukan dalam file konfigurasi [13].</span></p>
<ul style="list-style:none;"><li>

<h2><a name="bookmark18"></a><span class="font3" style="font-weight:bold;"><a name="bookmark19"></a>3.5. &nbsp;&nbsp;&nbsp;PRNG</span></h2></li></ul>
<p><span class="font3">Bilangan acak (</span><span class="font3" style="font-style:italic;">random number</span><span class="font3">) merupakan elemen penting dalam ilmu komputer. Pada dasarnya istilah bilangan acak berasal dari </span><span class="font3" style="font-style:italic;">Pseudo-random Number Generator</span><span class="font3"> (PRNG) yang merupakan algoritma perangkat lunak deterministik untuk menciptakan bilangan acak. Bilangan acak digunakan dalam berbagai bagian </span><span class="font3" style="font-style:italic;">cryptography</span><span class="font3">, </span><span class="font3" style="font-style:italic;">random sampling</span><span class="font3">, </span><span class="font3" style="font-style:italic;">Monte Carlo method</span><span class="font3">, </span><span class="font3" style="font-style:italic;">statistics</span><span class="font3">, </span><span class="font3" style="font-style:italic;">simulation &amp;&nbsp;modeling</span><span class="font3">, </span><span class="font3" style="font-style:italic;">lottery</span><span class="font3">, dan </span><span class="font3" style="font-style:italic;">random distribution</span><span class="font3"> [15].</span></p>
<ul style="list-style:none;"><li>

<h2><a name="bookmark35"></a><span class="font3" style="font-weight:bold;"><a name="bookmark36"></a>5. &nbsp;&nbsp;&nbsp;&nbsp;Kesimpulan</span></h2></li></ul>
<p><span class="font3">Berdasarkan kegiatan penelitian yang telah penulis lakukan, aplikasi yang dirancang pada penelitian dapat memperkuat proses autentikasi dalam memvalidasi komputer klien yang akan terhubung dengan sistem server menggunakan </span><span class="font3" style="font-style:italic;">random number sequence port</span><span class="font3">. </span><span class="font3" style="font-style:italic;">Random number sequence port</span><span class="font3"> diperoleh dari implementasi algoritma PRNG yang mana pembangkitan nilai </span><span class="font3" style="font-style:italic;">random</span><span class="font3"> menggunakan acuan </span><span class="font3" style="font-style:italic;">seed</span><span class="font3"> nilai </span><span class="font3" style="font-style:italic;">epochtime</span><span class="font3"> (waktu sistem) ditambah dengan kunci privat.
