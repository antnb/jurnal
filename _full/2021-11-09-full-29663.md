<h2><a name="bookmark4"></a><span class="font2" style="font-weight:bold;"><a name="bookmark5"></a>Abstrak</span></h2>
<p><span class="font2" style="font-style:italic;">Penelitian ini berjudul “Analisis Makna kontekstual di Peralihan Kode yang ditemukan di Film Check in Bangkok”. Alih kode dapat ditemukan dalam bentuk tulisan dan lisan baik di koran, majalah, buku, novel, program televisi, radio dan lainnya. Dalam kasus ini, hal itu dapat terjadi antara kalimat (inter-sentensial) atau dalam satu kalimat (intra-sentensial ) dan juga dapat dipengaruhi oleh situasi percakapan. Halliday (1985) menyatakan, penggunaan bahasa memiliki situasi konteks. Penelitian ini akan menganalisis konteks situasi yang mempengaruhi alih kode dalam sumber data.</span></p>
<p><span class="font2" style="font-style:italic;">Data penelitian diambil dari film Indonesia di YouTube yang berjudul “Checkin Bangkok”. Data yang telah dikumpulkan oleh studi pustaka dan metode dokumentasi, diklasifikasikan masing-masing jenis alih kode berdasarkan teori jenis alih kode yang diusulkan oleh Poplack, kemudian data juga dianalisis berdasarkan teori konteks situasi yang diusulkan oleh Holmes.</span></p>
<p><span class="font2" style="font-style:italic;">Hasil analisis dari penelitian ini menunjukkan bahwa tag, inter-sentensial dan intra-sentensial pada peralihan kode ditemukan dalam sumber data. Sementara itu, untuk konteks situasi faktor yang paling mempengaruhi pembicara untuk menggunakan alih kode adalah pembicara, latar/keadaan, topik dan fungsi/tujuan.</span></p>
<p><span class="font2" style="font-style:italic;">Kata kunci: alih kode, konteks situasi.</span></p>
<ul style="list-style:none;"><li>
<h2><a name="bookmark6"></a><span class="font2" style="font-weight:bold;"><a name="bookmark7"></a>1. &nbsp;&nbsp;&nbsp;Background of The Study</span></h2></li></ul>
<p><span class="font2">Most people in the world speak more than one language. As we know that language is an important aspect in life because language is a system of communication in which consist of sound, words and grammar. In this case, the people who master more than one language are called bilingual or multilingual. The condition where people conducting a communication tend to switch from one code to another, which involve words, phrases or involves several sentences, this is</span></p>
<p><span class="font2">called code switching (Grosjean, 1982:146). Code switching can occur in conversation between speakers’ turns or a single speakers’ turn. In the case, it can occur between sentences (inter-sentential) or within a single sentence (intra-sentential). According to the explanation above, the movie entitled </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> is chosen because there are many code switching found in the movie itself. It is because the theme in the movie uses the modern life style, as we know that the casts are well educated and one of the actors is from India.</span></p>
<ul style="list-style:none;"><li>

<p><span class="font2">Based on the background above the problems arise as follows:</span></p>
<ul style="list-style:none;"><li>
<p><span class="font2">1. &nbsp;&nbsp;&nbsp;What types of code switching are found in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie?</span></p></li>
<li>
<p><span class="font2">2. &nbsp;&nbsp;&nbsp;What factors of context of situation that influence the determining the choice of code switching found in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie?</span></p></li></ul>
<ul style="list-style:none;"><li>
<h2><a name="bookmark10"></a><span class="font2" style="font-weight:bold;"><a name="bookmark11"></a>3. &nbsp;&nbsp;&nbsp;Aims of Study</span></h2></li></ul>
<ul style="list-style:none;"><li>
<p><span class="font2">1. &nbsp;&nbsp;&nbsp;To classify and describe the types of code switching found in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie.</span></p></li>
<li>
<p><span class="font2">2. &nbsp;&nbsp;&nbsp;To determine and explain factors of context of situation that influence the code switching found in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie.</span></p></li></ul>
<ul style="list-style:none;"><li>
<h2><a name="bookmark12"></a><span class="font2" style="font-weight:bold;"><a name="bookmark13"></a>4. &nbsp;&nbsp;&nbsp;Research Method</span></h2></li></ul>
<p><span class="font2">The data of the study was taken from the Indonesian movie in YouTube entitled </span><span class="font2" style="font-style:italic;">“Check-in Bangkok”. Documentation method is used in collecting the data. </span><span class="font2">this study applied the descriptive qualitative research method. In this case, the theory of code switching that is proposed by Poplack (in Romaine 1989) was used to analyze the types of code switching. This study also used the theory context of situation that is proposed by Holmes (1992:12) to analyze the factors of contextual situation that influence the code switching which are found in the data.</span></p>
<ul style="list-style:none;"><li>
<h2><a name="bookmark14"></a><span class="font2" style="font-weight:bold;"><a name="bookmark15"></a>5. &nbsp;&nbsp;&nbsp;Findings and Discussion</span></h2></li></ul>
<p><span class="font2">The following examples presents the types of code switching and the factors that influence the speaker to use code switching</span></p>
<ul style="list-style:none;"><li>
<h2><a name="bookmark16"></a><span class="font2" style="font-weight:bold;"><a name="bookmark17"></a>5.1 &nbsp;&nbsp;&nbsp;Types of Code Switching</span><br><br><span class="font2" style="font-weight:bold;"><a name="bookmark18"></a>5.1.1 &nbsp;&nbsp;&nbsp;Tag Switching</span></h2></li></ul>
<p><span class="font2">Tag-switching is simply the insertion of a tag in one language in an utterance which is entirely in the other language, e.g. you know, I mean, right?. The examples of data analysis as follows.</span></p>
<p><span class="font2">Table of Data 1. (scene 9, line 99)</span></p>
<table border="1">
<tr><td style="vertical-align:bottom;">
<p><span class="font2">Papa</span></p></td><td style="vertical-align:bottom;">
<p><span class="font2">: </span><span class="font2" style="font-weight:bold;">Yes! </span><span class="font2" style="font-style:italic;">tapi dengan satu syarat.</span></p></td></tr>
</table>
<p><span class="font2">The type of code switching used in above sentence is considered into tag switching. The tag </span><span class="font2" style="font-weight:bold;">yes! </span><span class="font2">put in the beginning of the utterance then followed by the Indonesian utterance </span><span class="font2" style="font-style:italic;">“tapi dengan satu syarat”</span><span class="font2">. This word has same meaning in Indonesian word “ya”. In this case, according to oxford dictionary fourth edition (2008:518) the word yes means an exclamation which is an answer or a response to show you agree with an idea, a statement. So the speaker switch into English to show an expression of the character.</span></p>
<ul style="list-style:none;"><li>
<h2><a name="bookmark19"></a><span class="font2" style="font-weight:bold;"><a name="bookmark20"></a>5.1.2 &nbsp;&nbsp;&nbsp;Inter-sentential Switching</span></h2></li></ul>
<p><span class="font2">Inter-sentential switching is described as the switch between sentence boundaries, where one sentence is in one language and the other language. The examples of data analysis as follows.</span></p>
<p><span class="font2">Table of Data 2. (scene 3, line 15)</span></p>
<table border="1">
<tr><td style="vertical-align:middle;">
<p><span class="font2">Renata</span></p></td><td style="vertical-align:middle;">
<p><span class="font2">: </span><span class="font2" style="font-weight:bold;">I miss you too. </span><span class="font2" style="font-style:italic;">Enggak kerasa yah udah 2 tahun kita ldr-an.</span></p></td></tr>
</table>
<p><span class="font2">The code switching used by the speaker occurs between the sentences boundary. Which firstly, the speaker used English </span><span class="font2" style="font-weight:bold;">I miss you too </span><span class="font2">then continued by Indonesian </span><span class="font2" style="font-style:italic;">Enggak kerasa yah udah 2 tahun kita ldr-an.</span><span class="font2"> Therefore, the utterance above is categorized as inter sentential switching. In this case the sentence </span><span class="font2" style="font-weight:bold;">I miss you too </span><span class="font2">directly used to response the previous statement I miss you. It also used to express someone feeling.</span></p>
<ul style="list-style:none;"><li>
<h2><a name="bookmark21"></a><span class="font2" style="font-weight:bold;"><a name="bookmark22"></a>5.1.3 &nbsp;&nbsp;&nbsp;Intra-sentential Switching</span></h2></li></ul>
<p><span class="font2">Intra-sentential switching refers to the switching that occurs inside the same clause or sentence which then contains elements of both languages. The examples of data analysis as follows.</span></p>
<p><span class="font2">Table of Data 3. (scene 12, line 165)</span></p>
<table border="1">
<tr><td style="vertical-align:bottom;">
<p><span class="font2">Glenn</span></p></td><td style="vertical-align:bottom;">
<p><span class="font2">: </span><span class="font2" style="font-style:italic;">udah udah gak apa-apa lagian cuma</span><span class="font2" style="font-weight:bold;"> hand phone </span><span class="font2" style="font-style:italic;">doang.</span></p></td></tr>
</table>
<p><span class="font2">The utterance above is categorized into intra sentential, where the switching English phrase appears in the middle sentence. The beginning start from Indonesian </span><span class="font2" style="font-style:italic;">udah udah gak apa-apa lagian cuma</span><span class="font2"> and appears English phrase in the middle </span><span class="font2" style="font-weight:bold;">hand phone </span><span class="font2">then followed by Indonesian word </span><span class="font2" style="font-style:italic;">doang.</span><span class="font2"> The phrase </span><span class="font2" style="font-weight:bold;">hand phone </span><span class="font2">in the data above can be understood easily in the English form. People nowadays more known the word </span><span class="font2" style="font-style:italic;">hand phone</span><span class="font2"> than </span><span class="font2" style="font-style:italic;">telepon genggam</span><span class="font2"> which if it translated into Indonesian because the word </span><span class="font2" style="font-weight:bold;">hand phone </span><span class="font2">is common used in Indonesian. According to oxford dictionary, the phrase </span><span class="font2" style="font-weight:bold;">hand phone </span><span class="font2">means a mobile phone.</span></p>

<h2><a name="bookmark33"></a><span class="font2" style="font-weight:bold;"><a name="bookmark34"></a>6. &nbsp;&nbsp;&nbsp;Conclusion</span></h2></li></ul>
<p><span class="font2">Based on the analysis in the previous chapter, the result of this study is that there are so many code switching and also the social factor of context of situation which occur in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie.</span></p>
<p><span class="font2">This study analyzed the code switching and context of situation which influenced the speaker uses code switching in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie. The analysis of this study found in the </span><span class="font2" style="font-style:italic;">Check-in Bangkok</span><span class="font2"> movie applied all the types of</span></p>
<p><span class="font2">code switching those are; tag switching, inter-sentential and intra-sentential switching. The code switching that found the most in the movie is intra sentential switching, then inter sentential switching and the last is tag switching.</span></p>
<p><span class="font2">The context of situation in this study is analyzed based on four social factors those are; the participants: (who or whom?), the setting or social context of the interaction: (where), the topic: (what), the function: (why) and this study found all factors that used in the movie.</span></p>
