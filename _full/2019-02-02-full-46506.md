<p><span class="font5" style="font-weight:bold;font-style:italic;">ABSTRACT</span></p>
<p><span class="font5" style="font-style:italic;">Zero Inflated Generalized Poisson</span><span class="font5"> (ZIGP) </span><span class="font5" style="font-style:italic;">is a regression model used to analyze Poisson distributed discrete data which contains mostly zero and tends to experience overdispersion (varians value greater than the mean value). The purpose of this research is to find out the best model and the factors which influence the maternal mortality in Bali Province in year 2016 by using ZIGP regression model. The data used in this research was data from health profile Bali Province with the object totally 57 district rate data has proportion of zeros value more than 50% on the response variable. The analysis result of ZIGP data on maternal mortality cannot modeled using the ZIGP so ZIGP regression model became ZIP model . The best model which resulted from ZIP regression got one free variable which have significant impact towards the total number of maternal mortality. This significant variabel is the percentage of mother did visiting to K1</span><span class="font5"> (X<sub>1</sub>)</span><span class="font5" style="font-style:italic;">.</span></p>
<p><span class="font5" style="font-weight:bold;font-style:italic;">Keywords</span><span class="font5" style="font-style:italic;">: Maternal mortality, Overdispersion, Zero Inflated Generalized Poisson (ZIGP), Zero Inflated Poisson (ZIP).</span></p>
<h2><a name="bookmark2"></a><span class="font5" style="font-weight:bold;"><a name="bookmark3"></a>1. PENDAHULUAN</span></h2>
<p><span class="font5">Analisis regresi adalah salah satu model yang digunakan untuk mengetahui hubungan sebab akibat antar variabel, yaitu variabel bebas (</span><span class="font5" style="font-style:italic;">X</span><span class="font5">) dengan variabel respons (</span><span class="font5" style="font-style:italic;">Y</span><span class="font5">). Variabel respons dan variabel bebas memiliki hubungan yang dapat dinyatakan dalam persamaan matematis yaitu persamaan regresi. Persamaan regresi dapat berbentuk persamaan regresi linear maupun persamaan regresi non-linear. Persamaan regresi linear digunakan untuk menganalisis variabel respons yang berupa peubah acak kontinu dan mengikuti distribusi normal. Namun banyak ditemukan variabel respons yang tidak berdistribusi normal dan tidak linear dalam parameter. Untuk mengatasi hal tersebut dikembangkan </span><span class="font5" style="font-style:italic;">generalized linear model </span><span class="font5">(GLM). GLM digunakan sebagai perluasan model regresi umum dengan variabel responsnya berdistribusi keluarga eksponensial, meliputi distribusi normal, binomial, Poisson, binomial negatif, eksponensial, gamma, dan</span></p>
<p><span class="font5">invers normal (Myers </span><span class="font5" style="font-style:italic;">et al</span><span class="font5">.,2010).</span></p>
<p><span class="font5">Regresi </span><span class="font5" style="font-style:italic;">Poisson</span><span class="font5"> digunakan apabila variabel respons adalah peubah acak diskret. Analisis data menggunakan model regresi </span><span class="font5" style="font-style:italic;">Poisson</span><span class="font5"> harus memenuhi asumsi seperti nilai varians sama dengan rata-rata dari variabel respons tersebut atau </span><span class="font5" style="font-style:italic;">equidispersi</span><span class="font5"> (Myers </span><span class="font5" style="font-style:italic;">et al.,</span><span class="font5"> 2010). Namun, dalam kenyataannya sering terjadi pelanggaran asumsi tersebut, yaitu nilai variansnya lebih besar dari nilai rata-rata yang dinamakan </span><span class="font5" style="font-style:italic;">over-dispersi. Overdispersi</span><span class="font5"> pada regresi Poisson dapat dilihat dari nilai taksiran dispersi yaitu nilai devians dan </span><span class="font5" style="font-style:italic;">pearson chi square</span><span class="font5"> dibagi dengan derajat bebasnya bernilai lebih besar dari satu. Jika terjadi overdispersi pada data, maka regresi Poisson kurang akurat digunakan untuk analisis karena akan berdampak pada nilai </span><span class="font5" style="font-style:italic;">standard error</span><span class="font5"> menjadi lebih kecil dari nilai sesungguhnya </span><span class="font5" style="font-style:italic;">(underestimate)</span><span class="font5">, sehingga kesimpulan yang diperoleh menjadi tidak valid (McCullagh &amp;&nbsp;Nelder, 1989).</span></p>
<p><span class="font5">Salah satu penyebab terjadinya overdispersi yaitu adanya nilai nol yang berlebih pada variabel respon (</span><span class="font5" style="font-style:italic;">excess zeros</span><span class="font5">). Untuk mengatasi masalah overdispersi tersebut dapat digunakan beberapa model regresi antara lain model regresi </span><span class="font5" style="font-style:italic;">zero inflated poisson</span><span class="font5"> (ZIP) dengan proporsi nol sekitar 63,7%, dan </span><span class="font5" style="font-style:italic;">zero inflated generalized poisson</span><span class="font5"> (ZIGP) dengan proporsi nol 65,7%. Model regresi </span><span class="font5" style="font-style:italic;">zero inflated generalized poisson</span><span class="font5"> (ZIGP) merupakan perluasan dari model regresi </span><span class="font5" style="font-style:italic;">Poisson</span><span class="font5"> dan merupakan gabungan dari model regresi </span><span class="font5" style="font-style:italic;">zero inflated poisson</span><span class="font5"> (ZIP) dan model regresi </span><span class="font5" style="font-style:italic;">generalized poisson</span><span class="font5"> (GP) (Famoye dan Singh, 2006).</span></p>
<p><span class="font5">Studi kasus yang digunakan pada penelitian ini adalah Angka Kematian Ibu Hamil di Provinsi Bali tahun 2016. Data menunjukkan terjadi peristiwa kematian ibu hamil tahun 2016 sebanyak 18 ibu di 57 Kecamatan (Dinas Kesehatan Provinsi Bali, 2016). Tujuan dari penelitian ini adalah untuk menentukan model terbaik ZIGP serta untuk mengetahui faktor-faktor yang mempengaruhi kematian ibu hamil.</span></p>

<h2><a name="bookmark12"></a><span class="font5" style="font-weight:bold;"><a name="bookmark13"></a>(ZIP)</span></h2>
<p><span class="font5">Berdasarkan analisis regresi </span><span class="font5" style="font-style:italic;">Zero Inflated Generalized Poisson</span><span class="font5"> (ZIGP) disimpulkan bahwa data jumlah kematian ibu hamil (</span><span class="font5" style="font-style:italic;">Y</span><span class="font5">) tidak dapat dimodelkan dengan menggunakan model regresi </span><span class="font5" style="font-style:italic;">Zero Inflated Generalized Poisson</span><span class="font5"> (ZIGP). Dengan demikian model ZIGP akan tereduksi menjadi model </span><span class="font5" style="font-style:italic;">Zero Inflated Poisson</span><span class="font5"> (ZIP). Langkah selanjutnya adalah menganalisis model regresi ZIP dengan bantuan </span><span class="font5" style="font-style:italic;">software Rstudio.</span></p>
<p><span class="font5">Analisis regresi dengan ZIP akan menghasilkan dua model yakni </span><span class="font5" style="font-style:italic;">count model</span><span class="font5"> untuk memprediksi data cacah dan </span><span class="font5" style="font-style:italic;">zero inflated </span><span class="font5">model untuk memprediksi probabilitas terjadinya </span><span class="font5" style="font-style:italic;">zero response</span><span class="font5"> pada data. Pemodelan regresi ZIP pada kasus kematian ibu hamil di-</span></p>
<p><span class="font5">lakukan dengan memodelkan dari keseluruhan kombinasi variabel bebas dan dari modelmodel tersebut dipilih model yang signifikan dan memiliki nilai AIC terkecil.</span></p>
