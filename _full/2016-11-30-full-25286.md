The aim of this research is to model and forecast the number of tourist arrivals to Bali using transfer function model based on exchange rate USD to IDR from January 2009 to December 2015 Transfer function model is a multivariate time series model which can be used to identify the effect of the exchange rate to the number of tourist arrivals to Bali The first stage in transfer function modeling is identification of ARIMA model in exchange rate USD to IDR variable
