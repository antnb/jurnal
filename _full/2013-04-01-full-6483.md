<h1><a name="bookmark2"></a><span class="font7" style="font-weight:bold;"><a name="bookmark3"></a>ABSTRAK</span></h1>
<p><span class="font7" style="font-style:italic;">File sharing</span><span class="font7"> merupakan sebuah sistem di mana banyak pengguna dapat berbagi berkas dan sumber daya penyimpanan dengan konsep </span><span class="font7" style="font-style:italic;">client-server</span><span class="font7"> yang dijalankan melalui jaringan. Sebuah </span><span class="font7" style="font-style:italic;">File Server</span><span class="font7"> memungkinkan pengguna untuk berbagi informasi melalui jaringan tanpa harus secara fisik mengirim </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> dengan </span><span class="font7" style="font-style:italic;">floppy disk</span><span class="font7"> atau beberapa perangkat penyimpanan </span><span class="font7" style="font-style:italic;">eksternal</span><span class="font7"> lainnya. Adapun protokol maupun aplikasi yang sering digunakan untuk berbagi berkas (</span><span class="font7" style="font-style:italic;">file sharing</span><span class="font7">) adalah </span><span class="font7" style="font-style:italic;">Network File System</span><span class="font7"> (NFS), dan </span><span class="font7" style="font-style:italic;">Primary Domain Controller </span><span class="font7" style="font-weight:bold;">(</span><span class="font7">PDC) Samba.</span></p>
<p><span class="font7">NFS </span><span class="font7" style="font-style:italic;">(</span><span class="font7">N</span><span class="font7" style="font-style:italic;">etwork File System)</span><span class="font7"> adalah sebuah kumpulan protokol yang digunakan untuk mengakses beberapa sistem berkas melalui jaringan. Sedangkan Samba adalah program yang bersifat </span><span class="font7" style="font-style:italic;">open source</span><span class="font7"> yang menyediakan layanan berbagi berkas (</span><span class="font7" style="font-style:italic;">file service</span><span class="font7">) dan berbagi alat pencetak (</span><span class="font7" style="font-style:italic;">print service</span><span class="font7">). Dari kedua protokol tersebut, masing-masing memiliki kelebihan dan kekurangannya tersendiri. Pada penelitian ini, penulis sudah mencoba menganalisis perbandingan diantara keduanya dari segi kecepatan </span><span class="font7" style="font-style:italic;">transfer</span><span class="font7"> data, dari penelitian yang telah dilakukan didapatkan hasil yaitu kinerja dari file sharing </span><span class="font7" style="font-style:italic;">Network File System</span><span class="font7"> (NFS) lebih baik dibandingkan dengan kinerja </span><span class="font7" style="font-style:italic;">file sharing Samba</span><span class="font7">.</span></p>
<p><span class="font7">Kata kunci : </span><span class="font7" style="font-style:italic;">File Sharing</span><span class="font7">, NFS</span><span class="font7" style="font-style:italic;">,</span><span class="font7"> PDC Samba.</span></p>
<p><span class="font7" style="font-weight:bold;font-style:italic;">ABSTRACT</span></p>
<p><span class="font7" style="font-style:italic;">File sharing is a system where many users can share files and storage resources with client-server concepts which run through the network. A File Server allows users to share information over a network without having to physically send the file to a floppy disk or some other external storage devices. The protocols and applications are often used to share files (file sharing) is the Network File System</span><span class="font7"> (NFS), </span><span class="font7" style="font-style:italic;">and the Primary Domain Controller</span><span class="font7"> (PDC) </span><span class="font7" style="font-style:italic;">Samba.</span></p>
<p><span class="font7">NFS </span><span class="font7" style="font-style:italic;">(Network File System) is a set of protocols used to access multiple file systems over a network. While SAMBA is an open source program that provides file sharing services (file service) and share a printer device (print service). Of both these protocols, each has its own advantages and disadvantages. on this research, the authors have tried to analyze the comparison between the two in terms of speed of data transfer, from research conducted showed that the performance of file sharing Network File System</span><span class="font7"> (NFS) </span><span class="font7" style="font-style:italic;">is better than the performance of the Samba file sharing.</span></p>
<p><span class="font7" style="font-style:italic;">Keywords : File Sharing</span><span class="font7">, NFS, PDC Samba.</span></p>

<h1><a name="bookmark4"></a><span class="font7" style="font-weight:bold;"><a name="bookmark5"></a>1. &nbsp;&nbsp;&nbsp;PENDAHULUAN</span></h1></li></ul>
<p><span class="font7" style="font-style:italic;">File sharing</span><span class="font7"> merupakan sebuah sistem di mana banyak pengguna dapat berbagi berkas dan sumber daya penyimpanan dengan konsep </span><span class="font7" style="font-style:italic;">client-server</span><span class="font7"> yang dijalankan melalui jaringan.</span></p>
<p><span class="font7">Sistem </span><span class="font7" style="font-style:italic;">Client-Server</span><span class="font7"> adalah pembagian kerja antara </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> dan </span><span class="font7" style="font-style:italic;">client</span><span class="font7"> yg mengakses </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> dalam suatu jaringan. Sebuah </span><span class="font7" style="font-style:italic;">File Server</span><span class="font7"> memungkinkan pengguna untuk berbagi informasi melalui jaringan tanpa harus secara fisik mengirim </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> dengan </span><span class="font7" style="font-style:italic;">floppy disk</span><span class="font7"> atau beberapa perangkat penyimpanan </span><span class="font7" style="font-style:italic;">eksternal</span><span class="font7"> lainnya. Adapun protokol maupun aplikasi yang sering digunakan untuk berbagi berkas (</span><span class="font7" style="font-style:italic;">file sharing</span><span class="font7">) adalah </span><span class="font7" style="font-style:italic;">Network File System </span><span class="font7">(NFS), dan </span><span class="font7" style="font-style:italic;">Primary Domain Controller </span><span class="font7" style="font-weight:bold;">(</span><span class="font7">PDC) Samba.</span></p>
<p><span class="font7">NFS </span><span class="font7" style="font-style:italic;">(</span><span class="font7">N</span><span class="font7" style="font-style:italic;">etwork File System)</span><span class="font7"> atau dalam Bahasa Indonesia: “Sistem Berkas Jaringan” adalah sebuah kumpulan protokol yang digunakan untuk mengakses beberapa sistem berkas melalui jaringan. Sedangkan Samba adalah program yang bersifat </span><span class="font7" style="font-style:italic;">open source</span><span class="font7"> yang menyediakan layanan berbagi berkas (</span><span class="font7" style="font-style:italic;">file service</span><span class="font7">) dan berbagi alat pencetak (</span><span class="font7" style="font-style:italic;">print service</span><span class="font7">). Sebagai sebuah aplikasi </span><span class="font7" style="font-style:italic;">file server</span><span class="font7">, Samba mengizinkan berkas, alat pencetak, dan beberapa sumber daya lainnya agar dapat digunakan oleh banyak pengguna dalam</span></p>
<p><span class="font7">keluarga sistem operasi UNIX, dan mengizinkan interoperabilitas dengan sistem operasi </span><span class="font7" style="font-style:italic;">Windows.</span><span class="font7"> Samba dibuat berdasarkan protokol </span><span class="font7" style="font-style:italic;">Server Message Block</span><span class="font7"> (SMB), oleh Andrew Tridgell. Dari kedua protokol tersebut, masing-masing memiliki kelebihan dan kekurangannya tersendiri.</span></p>
<p><span class="font7">Melalui penelitian ini, penulis sudah mencoba menganalisis perbandingan diantara keduanya berdasarkan kecepatan </span><span class="font7" style="font-style:italic;">transfer</span><span class="font7"> data dan integritas data yang dikirim.</span></p>

<p><span class="font7" style="font-weight:bold;">2. &nbsp;&nbsp;&nbsp;TINJAUAN PUSTAKA</span></p>
<ul style="list-style:none;">
<li>
<p><span class="font7" style="font-weight:bold;">2.1</span><span class="font7" style="font-weight:bold;font-style:italic;"> &nbsp;&nbsp;&nbsp;Network File System</span><span class="font7" style="font-weight:bold;"> (NFS)</span></p></li></ul></li></ul>
<p><span class="font7">NFS adalah singkatan dari </span><span class="font7" style="font-style:italic;">Network File System</span><span class="font7">, yang pertama kali dikembangkan oleh </span><span class="font7" style="font-style:italic;">Sun Microsystems</span><span class="font7"> pada tahun 80an, sebagai sarana untuk berbagi </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> pada lingkungan kerja </span><span class="font7" style="font-style:italic;">diskless</span><span class="font7">. NFS menyediakan sarana untuk berbagi </span><span class="font7" style="font-style:italic;">file </span><span class="font7">antar jaringan, sehingga suatu mesin dapat mengakses </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> di mesin lain seolah-olah mengakses </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> sistem lokal. Protokol ini bekerja dengan sistem </span><span class="font7" style="font-style:italic;">client/server. Server </span><span class="font7">akan meng-</span><span class="font7" style="font-style:italic;">export</span><span class="font7"> suatu </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> sistem sehingga dapat di-</span><span class="font7" style="font-style:italic;">share</span><span class="font7"> di jaringan, selanjutnya, </span><span class="font7" style="font-style:italic;">user</span><span class="font7"> akan melakukan </span><span class="font7" style="font-style:italic;">mounting</span><span class="font7"> ke </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> tersebut.</span></p>
<p><span class="font7" style="font-style:italic;">Network File System</span><span class="font7"> (NFS) umumnya menggunakan protokol </span><span class="font7" style="font-style:italic;">Remote Procedure</span></p>
<p><span class="font7" style="font-style:italic;">Call</span><span class="font7"> (RPC) yang berjalan di atas UDP dan membuka </span><span class="font7" style="font-style:italic;">port</span><span class="font7"> UDP dengan nomor </span><span class="font7" style="font-style:italic;">port </span><span class="font7">2049 untuk komunikasi antara </span><span class="font7" style="font-style:italic;">client</span><span class="font7"> dan </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> di dalam jaringan. </span><span class="font7" style="font-style:italic;">client</span><span class="font7"> NFS selanjutnya akan menerima berkas dari </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> NFS, sementara </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> NFS mengirim berkas lokal kepada </span><span class="font7" style="font-style:italic;">client.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7" style="font-weight:bold;">7. 2.1.1 Arsitektur dan Komponen NFS</span></p></li></ul>
<p><span class="font7" style="font-weight:bold;">(</span><span class="font7" style="font-weight:bold;font-style:italic;">Network File System</span><span class="font7" style="font-weight:bold;">)</span></p>
<p><span class="font7" style="font-style:italic;">Network File System</span><span class="font7"> (NFS) adalah sebuah protokol yang berada pada </span><span class="font7" style="font-style:italic;">layer </span><span class="font7">aplikasi dari model TCP / IP. Sistem NFS bekerja meliputi lapisan sesi, presentasi dan aplikasi pada lapisan OSI </span><span class="font7" style="font-style:italic;">Reference Model</span><span class="font7">. Arsitektur dan Komponen Utama Pengoperasian NFS didefinisikan dalam bentuk tiga komponen utama yang dapat dipandang sebagai layanan yang berada di masing-masing dari tiga lapisan model OSI yang sesuai dengan TCP / IP </span><span class="font7" style="font-style:italic;">application layer</span><span class="font7"> (lihat Gambar 2.5). Komponen-komponen ini adalah:</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7">1.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;External Data Representation </span><span class="font7">(XDR): &nbsp;XDR adalah bahasa</span></p></li></ul>
<p><span class="font7">deskriptif yang memungkinkan tipe data harus didefinisikan secara konsisten.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7">2.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Remote Procedure Call</span><span class="font7">(RPC): RPC adalah lapisan sesi pada lapisan OSI layanan ini umumnya digunakan untuk</span></p></li></ul>
<p><span class="font7">mengimplementasikan </span><span class="font7" style="font-style:italic;">client / server</span><span class="font7"> fungsi </span><span class="font7" style="font-style:italic;">internetworking</span><span class="font7">.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7">3. &nbsp;&nbsp;&nbsp;NFS Prosedur dan Operasional: Fungsi sebenarnya dari NFS adalah diimplementasikan dalam bentuk prosedur dan fungsi operasi yang konseptual pada tujuh lapisan model OSI.</span></p><img src="https://jurnal.harianregional.com/media/6483-1.jpg" alt="" style="width:186pt;height:142pt;"></li></ul>
<p><span class="font7">Gambar 1. </span><span class="font7" style="font-style:italic;">NFS Architecture and Components</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7" style="font-weight:bold;">2.2</span><span class="font7" style="font-weight:bold;font-style:italic;"> &nbsp;&nbsp;&nbsp;Server Message Block</span><span class="font7" style="font-weight:bold;"> (SMB)</span></p></li></ul>
<p><span class="font7" style="font-style:italic;">Server Message Block</span><span class="font7"> disingkat SMB adalah </span><span class="font7" style="font-style:italic;">protocol client/server</span><span class="font7"> yang ditujukan sebagai layanan untuk berbagi berkas (</span><span class="font7" style="font-style:italic;">file sharing</span><span class="font7">) di dalam sebuah jaringan. Protokol ini sering digunakan di dalam sistem operasi </span><span class="font7" style="font-style:italic;">windows</span><span class="font7"> dan IBM. Sistem operasi berbasis UNIX juga dapat menggunakannya dengan tambahan perangkat lunak yang disebut dengan Samba.</span></p>
<p><span class="font7">SMB awalnya menggunakan protokol NetBios sebagai protokol di mana</span></p>
<p><span class="font7">ia berjalan, sebelum menggunakan protokol NetBios </span><span class="font7" style="font-style:italic;">over</span><span class="font7"> TCP/IP (NBT) sebagai protokol lapisan </span><span class="font7" style="font-style:italic;">transport</span><span class="font7">-nya. Dengan begitu, SMB juga dapat digunakan dalam sebuah jaringan TCP/IP yang lebih luas dukungannya. SMB mendukung fungsi-fungsi seperti:</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7">1. &nbsp;&nbsp;&nbsp;Membuka dan menutup koneksi antara </span><span class="font7" style="font-style:italic;">client</span><span class="font7"> SMB (yang disebut sebagai </span><span class="font7" style="font-style:italic;">Redirector</span><span class="font7">) dan </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> SMB (komputer yang menjalankan </span><span class="font7" style="font-style:italic;">file and print services</span><span class="font7">) untuk mengizinkan </span><span class="font7" style="font-style:italic;">client</span><span class="font7"> agar dapat mengakses sumber daya di dalam jaringan.</span></p></li>
<li>
<p><span class="font7">2. &nbsp;&nbsp;&nbsp;Menemukan, membaca, dan menulisi berkas dalam sebuah </span><span class="font7" style="font-style:italic;">file server.</span></p></li>
<li>
<p><span class="font7">3. &nbsp;&nbsp;&nbsp;Menemukan dan menggunakan alat pencetak yang digunakan bersama-sama untuk mencetak secara jarak jauh.</span></p></li></ul>
<ul style="list-style:none;"><li>
<h1><a name="bookmark6"></a><span class="font7" style="font-weight:bold;"><a name="bookmark7"></a>2.3 &nbsp;&nbsp;&nbsp;Samba</span></h1></li></ul>
<p><span class="font7">Samba merupakan serangkaian aplikasi unix yang berkomunikasi dengan protokol </span><span class="font7" style="font-style:italic;">Server Message Block</span><span class="font7"> (SMB), yang dikembangkan oleh Andrew Tridgell yang selanjutnya sampai saat ini banyak dibantu oleh para </span><span class="font7" style="font-style:italic;">programmer</span><span class="font7"> di </span><span class="font7" style="font-style:italic;">internet </span><span class="font7">dengan tetap di supervisi oleh Andrew Tridgell. SMB adalah protokol komunikasi data yang juga digunakan oleh </span><span class="font7" style="font-style:italic;">Microsost </span><span class="font7">dan OS/2 untuk menampilkan fungsi jaringan </span><span class="font7" style="font-style:italic;">client-server</span><span class="font7"> yang menyediakan</span></p>
<p><span class="font7" style="font-style:italic;">sharing file</span><span class="font7"> dan </span><span class="font7" style="font-style:italic;">printer</span><span class="font7"> serta tugas-tugas lainya yang berhubungan. </span><span class="font7" style="font-style:italic;">Samba server </span><span class="font7">memungkinkan &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;komputer-komputer</span></p>
<p><span class="font7" style="font-style:italic;">Unix/Linux</span><span class="font7"> melakukan </span><span class="font7" style="font-style:italic;">sharing file</span><span class="font7"> dan </span><span class="font7" style="font-style:italic;">printer</span><span class="font7"> dengan komputer </span><span class="font7" style="font-style:italic;">windows</span><span class="font7"> bukan hanya dengan komputer-komputer Unix/Linux sendiri. Beberapa fungsi yang disediakan Samba </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> yaitu sebagai berikut :</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7">a.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Sharing</span><span class="font7"> file/direktori antar Unix/Linux dengan </span><span class="font7" style="font-style:italic;">windows client</span><span class="font7">.</span></p></li>
<li>
<p><span class="font7">b.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Sharing printer</span><span class="font7"> pada Samba </span><span class="font7" style="font-style:italic;">server </span><span class="font7">dengan </span><span class="font7" style="font-style:italic;">windows client.</span></p></li>
<li>
<p><span class="font7">c. &nbsp;&nbsp;&nbsp;Memudahkan proses </span><span class="font7" style="font-style:italic;">network browsing.</span></p></li>
<li>
<p><span class="font7">d. &nbsp;&nbsp;&nbsp;Menyediakan proses Autentikasi komputer </span><span class="font7" style="font-style:italic;">windows client</span><span class="font7"> ketika </span><span class="font7" style="font-style:italic;">login</span><span class="font7"> ke </span><span class="font7" style="font-style:italic;">Windows domain</span></p></li>
<li>
<p><span class="font7">e. &nbsp;&nbsp;&nbsp;Menyediakan dan membantu proses </span><span class="font7" style="font-style:italic;">netbios name resolution</span><span class="font7"> dengan </span><span class="font7" style="font-style:italic;">Windows Internet.</span></p></li>
<li>
<p><span class="font7">f.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;Name Service</span><span class="font7"> (WINS) </span><span class="font7" style="font-style:italic;">name-server resolution.</span></p></li></ul>
<p><span class="font7">Samba merupakan serangkaian aplikasi yang juga meliputi aplikasi </span><span class="font7" style="font-style:italic;">client </span><span class="font7">maka Samba juga menyediakan </span><span class="font7" style="font-style:italic;">tools client </span><span class="font7">untuk memungkinkan </span><span class="font7" style="font-style:italic;">user</span><span class="font7"> pada sistem </span><span class="font7" style="font-style:italic;">Unix/Linux</span><span class="font7"> mengakses direktori dan </span><span class="font7" style="font-style:italic;">printer </span><span class="font7">yang terdapat pada sistem </span><span class="font7" style="font-style:italic;">windows</span><span class="font7"> dan </span><span class="font7" style="font-style:italic;">Samba server</span><span class="font7"> melalui jaringan. Selain </span><span class="font7" style="font-style:italic;">tools client</span><span class="font7">, pada Samba juga terdapat beberapa</span></p>
<p><span class="font7">aplikasi </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> diantaranya yaitu </span><span class="font7" style="font-style:italic;">smbd</span><span class="font7"> dan </span><span class="font7" style="font-style:italic;">nmbd.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7">a.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;smbd</span><span class="font7"> adalah aplikasi </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> atau </span><span class="font7" style="font-style:italic;">daemon</span><span class="font7"> yang menangani proses </span><span class="font7" style="font-style:italic;">sharing</span><span class="font7"> file/direktori dan </span><span class="font7" style="font-style:italic;">printer</span><span class="font7">, juga menangani proses autentikasi dan otorisasi dengan SMB </span><span class="font7" style="font-style:italic;">client.</span></p></li>
<li>
<p><span class="font7">b.</span><span class="font7" style="font-style:italic;"> &nbsp;&nbsp;&nbsp;nmbd</span><span class="font7"> adalah aplikasi </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> atau </span><span class="font7" style="font-style:italic;">daemon</span><span class="font7"> yang mendukung Netbios name </span><span class="font7" style="font-style:italic;">service</span><span class="font7"> dan WINS , yang juga membantu proses </span><span class="font7" style="font-style:italic;">network browsing </span><span class="font7">pada </span><span class="font7" style="font-style:italic;">windows client.</span></p></li></ul>
<p><span class="font7">Dengan memanfaatkan Samba, </span><span class="font7" style="font-style:italic;">server </span><span class="font7">Linux dapat berfungsi sebagai </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> SMB. </span><span class="font7" style="font-style:italic;">Client Windows</span><span class="font7"> dapat mengakses </span><span class="font7" style="font-style:italic;">server linux</span><span class="font7"> layaknya mengakses </span><span class="font7" style="font-style:italic;">server Windows. </span><span class="font7">Samba juga dapat berbagi </span><span class="font7" style="font-style:italic;">file</span><span class="font7"> ataupun </span><span class="font7" style="font-style:italic;">printer,</span><span class="font7"> berfungsi sebagai PDC dan lain sebagainya.</span></p>
<ul style="list-style:none;"><li>
<p><span class="font7" style="font-weight:bold;">2.4</span><span class="font7" style="font-weight:bold;font-style:italic;"> &nbsp;&nbsp;&nbsp;Wireshark</span></p></li></ul>
<p><span class="font7" style="font-style:italic;">Wireshark</span><span class="font7"> adalah program yang berfungsi untuk mengetahui kejadian yang terjadi pada saat kita melakukan interaksi dengan </span><span class="font7" style="font-style:italic;">internet</span><span class="font7">. <p><span class="font7" style="font-style:italic;">Gambar 5. Grafik kinerja (kecepatan transfer) masing-masing server dengan 2 client</span></p>

Pada server Samba terlihat rentang kecepatan rata-rata pada 63,969 Mbit/sec sampai 81,674 Mbit/sec, sedangkan pada </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> NFS terlihat rentang kecepatan rata-ratanya pada 84,172 Mbit/sec sampai 95,016 Mbit/sec. Jadi setelah melakukan pengujian sebanyak 10 kali dimana masing-masing </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> dengan 2 </span><span class="font7" style="font-style:italic;">client</span><span class="font7"> yang mengaksesnya, didapatkan data bahwa kinerja </span><span class="font7" style="font-style:italic;">server</span><span class="font7"> NFS lebih baik</span></p>
<p><span class="font7">dalam segi kecepatan daripada </span><span class="font7" style="font-style:italic;">server </span><span class="font7">Samba.</span></p>

<h1><a name="bookmark18"></a><span class="font7" style="font-weight:bold;"><a name="bookmark19"></a>5. &nbsp;&nbsp;&nbsp;KESIMPULAN</span></h1></li></ul>
<p><span class="font7">Kesimpulan yang didapat dari penelitian yang telah kami lakukan adalah kinerja dari file sharing </span><span class="font7" style="font-style:italic;">Network File System</span><span class="font7"> (NFS) lebih baik dari kinerja file sharing Samba dari segi kecepatan. Hal ini dapat dilihat dari kecepatan rata-rata yang kami peroleh dari data hasil pengujian. Dalam pengujian dengan satu client, server NFS memilki kecepatan ratarata antara 79,894 Mbit/sec sampai 92.826 Mbit/sec.</span></p>
<p><span class="font7">Sedangkan server Samba memilki kecepatan rata-rata antara 49,352 Mbit/sec sampai 57,831 Mbit/sec. Dalam pengujian dengan dua client, server NFS memilki kecepatan rata-rata antara 84,172 Mbit/sec sampai 95,016 Mbit/sec. Sedangkan </span><span class="font7" style="font-style:italic;">server </span><span class="font7">Samba memiliki kecepatan rata-rata antara 63,969 Mbit/sec sampai 81,674 Mbit/sec.</span></p>
