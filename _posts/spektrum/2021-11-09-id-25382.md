## Abstract:
Perkembangan dan persaingan agen properti terus meningkat. PT. Grha Kusuma sebagai agen properti dalam proses kerjanya masih manual. Dengan permasalahan tersebut, dilakukan penelitian sistem informasi agen properti berbasis web dengan menggunakan bahasa pemrograman PHP, DBMS MySQL dan Bootstrap (framework HTML, CSS dan Javascript). Metode penelitian dengan melakukan kajian pustaka, pengumpulan data, pemodelan sistem, desain aplikasi, pemrograman aplikasi dan pengujian aplikasi dengan metode black box. Penelitian menghasilkan aplikasi agen properti berbasis web yang dapat dijalankan dengan baik pada browser mode mobile maupun desktop. Dengan aplikasi ini, agen properti dapat memberikan layanan jasanya dengan cepat, tepat dan dapat diakses di mana saja.
