## Abstract:
"Pada penelitian ini, unjuk kerja kode hybrid orthogonal small set dengan panjang 16 chip dievaluasi. Hasil penelitian menunjukkan bahwa auto-correlation function (ACF) and cross-correlation function (CCF) kode hybrid hampir sama dengan kode existing orthogonal small set Kasami. Kode existing orthogonal small set Kasami memiliki unjuk kerja BER (bit error rate) sedikit lebih baik dari pada kode hybrid orthogonal small set pada kondisi kanal Rayleigh frequency selective fading dalam sistem synchronous CDMA. Namun, kode hybrid orthogonal small set memiliki jumlah kode lebih banyak dalam chip set, karenanya dapat menampung lebih banyak user pada suatu base station. Sehingga, kode hybrid orthogonal small set lebih tepat untuk diterapkan dalam sistem synchronous CDMA. DOI: 10.24843/MITE.1501.04"
