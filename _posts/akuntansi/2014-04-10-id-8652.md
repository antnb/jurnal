## Abstract:
"ABSTRAK Penelitian ini menguji pengaruh return on equity, dividend payout ratio, dan price to earnings ratiopada return saham di Bursa Efek Indonesia, yangdifokuskan pada perusahaan yang terdaftar dalam indeks LQ45tahun 2010-2012.Menggunakan data sekunderberupa laporan keuangan tahunan perusahaan. Hasil proses seleksi sampel memperoleh 105 sampel, yang terdiri dari 47 perusahaan dan dianalisis menggunakan analisis regresi linier berganda.Hasil analisis menunjukkan variabel ROE dan DPR berpengaruh positif dan signifikan pada return saham, sedangkan variabel PER tidak berpengaruh pada return saham. Kata kunci: DPR, PER, return saham, ROE"
