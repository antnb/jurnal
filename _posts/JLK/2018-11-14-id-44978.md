## Abstract:
"Dalam dunia perkantoran, pegawai memiliki kewajiban untuk melaksanakan tugas atau pekerjaan di kantor. Setiap harinya, pegawai melaksanakan beberapa tugas yang harus diselesaikan. Untuk memberi transparansi kepada atasan, maka arsip kegiatan harian pegawai sangat diperlukan. Untuk itu dibuatlah sebuah Sistem Informasi Manajemen Kepegawaian (SIMPEG) berbasis web untuk menyimpan arsip aktivitas harian secara online. Sistem ini memudahkan user yaitu pegawai dalam melakukan input aktivitas harian secara online, tanpa harus membuka file excel di komputer setiap harinya. User dapat menginput aktivitas harian melalui browser HP. Sistem ini dirancang dengan metode pengembangan system Extreme Programming (XP). Pengujian yang dilakuakan terhadap sistem yaitu menggunakan metode Black box yang menunjukkan fungsionalitas sistem."
