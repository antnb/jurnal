## Abstract:
"ABSTRACT The purpose of this study was to determine the effect of leadership style on library promotion in the Bali Provincial Archives and Library Service. This study uses descriptive methods and quantitative approaches. Data collection techniques using questionnaire and literature study methods with sampling techniques using nonprobability sampling as many as 87 respondents. Data processing techniques use computational calculations SPSS version 25. The results showed that the leadership style applied in the Archives and Library Service of Bali Province had an effect on promotional activities. Regression analysis results obtained Y = 0.737 + 0.729X From the regression equation above, it can be seen that the magnitude of X1 is 0.729 with a significance level of 0.000 and the amount of R square is 0.537. Keywords: Leadership Style, Library Promotion, Library Head, Librarian"
