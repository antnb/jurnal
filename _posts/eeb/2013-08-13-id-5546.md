## Abstract:
"Penelitian ini bertujuan untuk mengetahui pengaruh parsial maupun simultan Economic Value Added, Market Value Added, Return On Asset dan Return On Equity terhadap return saham pada perusahaan perbankan yang terdaftar di Bursa Efek Indonesia. Penentuan sampel dilakukan dengan metode purposive sampling, dan diperoleh sebanyak 9 perusahaan perbankan sebagai sampel penelitian. Pengujian hipotesis penelitian digunakan teknik analisis regresi linear berganda. Hasil penelitian menunjukkan bahwa Economic Value Added, Return On Asset, Return On Equity tidak berpengaruh secara signifikan pada return saham. Sedangkan Market Value Added berpengaruh positif signifikan pada return saham. Secara simultan keempat variabel penelitian menunjukkan pengaruh signifikan terhadap return saham. Disarankan untuk penelitian selanjutnya dapat menambahkan variabel likuiditas, variabel aktivitas. Selain itu juga dapat dipertimbangkan faktor-faktor makro ekonomi sebagai faktor-faktor yang dapat mempengaruhi return saham. Kata kunci: return saham, economic value added (EVA), market value added (MVA), return on asset (ROA) dan return on equity (ROE)"
